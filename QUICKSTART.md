# 快速启动指南

## 1. 安装依赖（如果还没安装）

```bash
npm install
```

## 2. 启动开发服务器

```bash
npm run dev
```

## 3. 访问应用

打开浏览器访问：http://localhost:3000

## 功能说明

### 首页功能
- **搜索**：支持按公司名称、行业、失败原因等搜索
- **筛选**：可按行业、失败原因、国家、年代筛选
- **排序**：支持按亏损金额、融资总额、存活时间、关闭时间排序
- **统计面板**：显示累计亏损、总融资额、平均存活时间等关键指标
- **数据可视化**：行业分布、失败原因统计、国家分布图表

### 详情页功能
- **完整信息**：展示项目的所有详细信息
- **深度分析**：详细的失败过程分析
- **失败原因**：多维度失败原因分析
- **经验教训**：可借鉴的经验总结
- **关联信息**：投资者、创始人、标签等

## 添加新案例

编辑 `data/startups.ts` 文件，按照现有格式添加新的失败案例数据。

## 自定义样式

- 主题颜色在 `tailwind.config.js` 中配置
- 全局样式在 `app/globals.css` 中定义

## 构建生产版本

```bash
npm run build
npm start
```
